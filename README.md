# pyyingyu
Key Features

    provides translate of python

Installation

    use pip:

        $ pip install pyyingyu

    use source code:

        $ python setup.py install

Getting Started

    an usage example of pyyingyu:
    
        import pyyingyu
        bd = BaiduTranslate()
        bd.translate('I love reading the book very much!','en','zh')
        'en' means English,and 'zh' means Chinese
